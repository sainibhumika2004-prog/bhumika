Informed Consent

Before using CogniTrace, please review and confirm your understanding:

What CogniTrace does
Records weekly observations from caregivers about their elderly family member's cognitive state. Computes risk scores using validated clinical instruments (AD8, IQCODE-16, DLBSQ).
Data handling (DPDP Act compliant)
  • Stored on Indian servers (AWS Mumbai region)
  • End-to-end encryption · audit logged
  • Only your linked doctor can access
  • HIPAA-equivalent safeguards
  • You can export or delete anytime
Important limitations
CogniTrace is a screening and monitoring tool, not a diagnostic device. Clinical diagnosis requires evaluation by a qualified neurologist. False positives and false negatives are possible.

# Production endpoint
https://api.cognitrace.in/v1

curl -H "Authorization: Bearer <token>" \
     https://api.cognitrace.in/v1/cohort
Python — Fetch a Cohort
import requests, pandas as pd

token = requests.post("https://api.cognitrace.in/oauth/token",
    data={"grant_type": "client_credentials"}).json()["access_token"]

resp = requests.get(
    "https://api.cognitrace.in/v1/cohort",
    headers={"Authorization": f"Bearer {token}"},
    params={"stage": "MCI_Likely,Probable_AD",
            "min_weeks": 12, "age_min": 60}
)
df = pd.DataFrame(resp.json()["participants"])
print(f"Matched {len(df)} participants")
R — Longitudinal Mixed-Effects
library(httr); library(dplyr); library(lme4)

df <- GET("https://api.cognitrace.in/v1/longitudinal",
          add_headers(Authorization = paste("Bearer", token))) %>%
      content("parsed") %>% as.data.frame()

# Mixed-effects model: decline by stage
model <- lmer(ad8_score ~ week * stage + (1 | participant_id), data = df)
summary(model)

Ethics & IRB
Compliance for research use
Data Anonymization Pipeline
  1. Direct identifier removal: Names, phone numbers, addresses, doctor codes stripped.
  2. Date shifting: Timestamps shifted ±30 days per participant while preserving relative order.
  3. K-anonymity (k=5): Age binned, rare districts aggregated.
  4. Differential privacy: Laplace mechanism with ε=0.5 for group sizes <100.
